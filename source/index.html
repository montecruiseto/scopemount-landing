  • Scopemount

    Scopemount is a suite of 21 beautiful themes for Telescope, an open source Meteor app that makes it easy to launch communities.

    Browse themes by pressing Prev" and "Next" at the bottom left of your screen. On desktop, you can use your keyboard arrows to navigate through the themes.

  • Design

    Laval is a colourful book club theme, based on the design of Books and Conversations. The background colours of each post are randomly generated on each page load.

    Demo

    Install and Setup

    You can install Laval automagically through Atmosphere by running the following command inside your Telescope directory:

    $ meteor add montecruiseto:telescope-theme-sm-laval

    If you want to customize the theme you can download the source, extract the Laval theme folder to your Telescope package directory and run the same command as above. Telescope will look for your local package first and use that one.

    To modify the colours you can change the parameters in post_item_laval.js, located in the scripts folder.

    Template.post_item.onRendered(function() {
    $(this.find('.laval-post')).css('background', randomColor({
        luminosity: 'bright',
        hue: 'green'
    }));
});

    Note: You can pass light, bright, dark or random to the luminosity property, and monochrome, red, orange, yellow, green, blue, purple or pink to the hue property.

    Name

    This theme was named after the Laval LMT, a telescope with a rotating 2.7-meter diameter mirror and a surface of liquid mercury, located in Vancouver, Canada.

    You can install Magellan automagically through Atmosphere by running the following command inside your Telescope directory:

    $ meteor add montecruiseto:telescope-theme-sm-magellan

    If you want to customize the theme you can download the source, extract the Magellan theme folder to your Telescope package directory and run the same command as above. Telescope will look for your local package first and use that one.

    For this theme, the function of "sticky" posts has been utilized to create "featured posts" which are permanently visible in the right sidebar.

    This theme was named after the Giant Magellan Telescope, a ground-based extremely large telescope in Chile, planned for completion in 2025.

  • Design

    Starfire gives you a base to create your very own gaming news site in no time. Go head-on with sites like Polygon, Metro and GamesRadar.

    Demo

    Install and Setup

    Set your Telescope admin settings to post 'grid' mode and include Embedly thumbs with a width of 340px and height of 192px.

    You can install Starfire automagically through Atmosphere by running the following command inside your Telescope directory:

    $ meteor add montecruiseto:telescope-theme-sm-starfire

    If you want to customize the theme you can download the source, extract the Starfire theme folder to your Telescope package directory and run the same command as above. Telescope will look for your local package first and use that one.

    Note 1: The theme makes use of the gooey button effect created by the talented Lucas Bebber. The buttons combine CSS3 transforms and transitions with a svg filter underneath. Although they work perfectly in Google chrome, they don't always render consistently in Safari. If anyone can solve that bug you'll be doing humanity a great favour. Until then, I would suggest to exclude the svg filter and retain only the CSS3 animation in your production app.

    Note 2: The font "Voltaire" is only used on the logo. If you are concerned with page load times, feel free to throw out one or more of the webfonts and use standard native fonts by editing "_typography.less".

    Name

    This theme was named after the Starfire Optical Range, a United States Air Force research laboratory in Albuquerque, New Mexico. Its has a 3.5 meter telescope which is "one of the largest telescopes in the world equipped with adaptive optics designed for satellite tracking".

  • Design

    Galileo is a modern Q&A theme, taking functional and visual cues from Quora and Medium. The theme ships with a colourful sunburst gradient on the side-nav. However, if you opt to install the package locally, you can edit the _colors.less file and switch to a cool aquamarine variant by uncommenting the corresponding code.

    Demo

    Install and Setup

    Note: Remember to set Telescope to 'side' navigation and post 'list' mode in your admin settings.

    You can install Galileo automagically through Atmosphere by running the following command inside your Telescope directory:

    $ meteor add montecruiseto:telescope-theme-sm-galileo

    If you want to customize the theme you can download the source, extract the Galileo theme folder to your Telescope package directory and run the same command as above. Telescope will look for your local package first and use that one.

    Out-of-the box Galileo includes 3 Google webfonts, which are mostly for showcasing purposes. The font "Yesteryear" is only used on the logo. If you are concerned with page load times, feel free to throw out one or more of the webfonts and use standard native fonts by editing "_typography.less".

    Name

    This theme was named after the Telescopio Nazionale Galileo, a 3.58m Italian telescope located on the island of La Palma in the Canary Islands archipelago.
