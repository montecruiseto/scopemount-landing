  • Laval Theme

    Turn your Telescope app into a blog with Magellan, Scopemount theme number nine. This particular theme was heavily inspired by the TechInAsia-blog design. The function of "sticky" posts has been utilized to create "featured posts" which are permanently visible in the right sidebar.

    Install and Setup

    You can install Magellan automagically through Atmosphere

    $ meteor add montecruiseto:telescope-theme-sm-magellan

    Alternatively if you want to customize the theme: Download and extract the theme folder to your Telescope package directory and run the same command as above. Telescope will look for the local package first and use that one.

    Note: Set your Telescope admin settings to post 'list' mode and include Embedly thumbs with a width of 200px and height of 112px.

    Name

    This theme was named after the Giant Magellan Telescope a ground-based extremely large telescope in Chile, planned for completion in 2025. The telescope is expected to have over five to ten times the light-gathering ability of existing ones. If you want to be notified of theme updates or new releases, you can leave your email on the The first page. Talk back next week. In the meantime, keep aiming for the stars.
