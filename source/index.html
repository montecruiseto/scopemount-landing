  • Theme

    Laval is a colourful book club theme, based on the design of Books and Conversations. The background colours of each post are randomly generated on each page load using randomColor.js.

    Demo

    Install and Setup

    You can install Laval automagically through Atmosphere by running the following command inside your meteor directory:

    $ meteor add montecruiseto:telescope-theme-sm-magellan

    If you want to customize the theme you can download the source, extract the Laval theme folder to your Telescope package directory and run the same command as above. Telescope will look for your local package first and use that one.

    Name

    This theme was named after the Laval LMT, a telescope with a rotating 2.7-meter diameter mirror and a surface of liquid mercury, located in Vancouver, Canada.
